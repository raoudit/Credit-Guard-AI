---

## 📌 Overview

CreditGuard AI is a machine learning-based fraud detection application that analyzes credit card transactions and identifies potentially fraudulent activities.

The application provides:
- Fraud risk prediction
- Transaction analysis
- Interactive visualizations
- Fraud detection reports

Built with Python and Streamlit, this project demonstrates how machine learning models can be integrated into an interactive web application for real-world financial security use cases.

---

# 🚀 Features

✅ Detect fraudulent credit card transactions  
✅ Generate fraud probability/risk scores  
✅ Classify transactions as Safe or Fraud  
✅ Interactive transaction visualizations  
✅ Upload custom CSV transaction data  
✅ Demo dataset for instant testing  
✅ Download fraud detection reports as CSV

# 📖 How To Use

1. Open the live application:
   
   https://credit-guard-ai-n3ket4egiifcjtbds5dxka.streamlit.app/

2. Click **"Try Demo Data"** to test the application instantly.

3. Upload your own credit card transaction CSV file.

4. The AI model analyzes transactions and predicts fraud risk.

5. View:
   - Fraud/Safe classification
   - Risk scores
   - Transaction analysis charts

6. Download the fraud detection report as a CSV file.
